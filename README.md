# you-tube-rag-langchain
YouTube RAG Chatbot built with LangChain, FAISS, Hugging Face Embeddings, and Google Gemini. The application extracts YouTube video transcripts, creates a vector database, and enables users to ask questions about video content using Retrieval-Augmented Generation (RAG)


# YouTube RAG using LangChain

A Retrieval-Augmented Generation (RAG) application that allows users to chat with YouTube videos.

## Features

- Extract YouTube transcripts
- Split text into chunks
- Generate embeddings using Sentence Transformers
- Store vectors using FAISS
- Answer questions using Gemini

## Tech Stack

- LangChain
- FAISS
- Gemini
- Sentence Transformers
- YouTube Transcript API

## Installation

```bash
pip install -r requirements.txt
```

## Run

Open the notebook and execute all cells.
